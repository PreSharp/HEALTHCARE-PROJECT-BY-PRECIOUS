# Healthcare Performance Dashboard

## Project Overview

The Healthcare Performance Dashboard is an Excel-based analytics project designed to monitor and evaluate key healthcare metrics across different departments, patient groups, and service areas. The dashboard provides a comprehensive view of healthcare performance, helping stakeholders identify trends, improve operational efficiency, and enhance patient outcomes.

This project transforms raw healthcare data into meaningful insights through interactive visualizations and performance indicators.


## Project Objectives

* Monitor overall healthcare performance.
* Track patient admission and discharge trends.
* Evaluate departmental performance.
* Analyze patient satisfaction levels.
* Identify operational bottlenecks and improvement opportunities.
* Support data-driven healthcare decision-making.
